Custom Cipher Encoder / Decoder

Use this tool to encode / decode text with a custom cipher! The total cipher consists of a tabular transposition cipher as well as a Vigenere cipher. For more details, the breakdown is available at this github repository . Happy ciphering!

To use, please input two keywords as well as the plain text you would like to encode / decode. The first keyword will be used for the tabular transformation encoding / decoding, and the second keyword will be used for the Vigenere cipher encoding / decoding. It currently only works on letters only.
